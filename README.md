# ☕ Chaiholic Cafe Website

A modern, responsive landing page designed for the cafe **Chaiholic**.  
This project focuses on creating a visually appealing and user-friendly café website with essential business sections.

## 🚀 Features
- Hero section with call-to-action buttons
- About section with brand story
- Featured menu & full food menu
- Ambience / gallery showcase
- Testimonials & social proof
- Location with Google Maps link
- Table reservation form with confirmation message
- Fully responsive design for mobile & desktop

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript (Form handling & interactions)

## 🎯 Purpose
This project was built as a frontend portfolio project to demonstrate:
- UI/UX design skills
- Responsive layout development
- Real business website structure
- Clean component-based sections

## 🌐 Future Improvements
- Backend reservation system
- WhatsApp booking integration
- Online ordering feature
- Admin menu management
